# nieuwerepo

Solana devnet test repository for small experiments and notes.

## What this is

A lightweight place to test devnet flows before using them in a real project.

## Planned checks

- Solana CLI setup
- devnet wallet balance
- airdrop behavior
- simple RPC requests
- basic TypeScript scripts later

## Quick commands

```bash
solana config set --url devnet
solana address
solana airdrop 2
solana balance
```

Devnet SOL is only for testing.
