# Solana Devnet Notes

## Setup

```bash
solana --version
solana config set --url devnet
solana config get
```

## Wallet

```bash
solana address
solana balance
solana airdrop 2
```

## Notes

- Use devnet for testing only.
- Do not reuse real wallets for quick experiments.
- Airdrop limits can happen. Waiting and retrying is normal.
- Keep private keys out of git.
